GPS Guided Robot

A modified hobby grade RC car with dedicated steering servo and eletronic speed control. Given a set of longitude and latitude coordinates, the robot will move to the location and drives back to the starting position. Built with Arduino Nano, U-blox NEO-6M GPS and coded in C/C++ using arduino IDE and libraries.
